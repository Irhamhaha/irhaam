Fungsi adalah organisasi yang baik, dapat digunakan kembali, digunakan untuk mencapai segmen kode tunggal, atau fungsi yang terkait. Fungsi dapat diterapkan untuk meningkatkan modularitas dan menggunakan kembali kode.

Contoh fungsi sederhananya seperti berikut

contoh

# Ini adalah fungsi
def hello():
    print("Hai")

# Dan ini untuk memanggil fungsi
hello()

Hasilnya

Fungsi panggilan

Mendefinisikan fungsi: untuk memberikan fungsi nama, menentukan parameter dari fungsi mengandung, dan struktur kode blok.

contoh

def hello(nama):
    print (nama)

hello("Irham")
